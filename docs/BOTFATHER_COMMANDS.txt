📝 КОМАНДЫ ДЛЯ РЕГИСТРАЦИИ В BOTFATHER
═══════════════════════════════════════════════════════════════════

Чтобы зарегистрировать команды в BotFather:

1. Открыть чат с @BotFather в Telegram
2. Отправить команду: /setcommands
3. Выбрать своего бота
4. Скопировать и вставить список команд ниже:

───────────────────────────────────────────────────────────────────
КОПИРОВАТЬ ОТСЮДА ↓
───────────────────────────────────────────────────────────────────

start - 🚀 Начать работу с ботом
help - ❓ Справка по использованию
search - 🔍 Обычный поиск вакансий
smart - 🧠 Умный поиск с AI (в свободной форме)
stats - 📊 Ваша статистика
calc - 🔢 Калькулятор

───────────────────────────────────────────────────────────────────
КОПИРОВАТЬ ДО СЮДА ↑
───────────────────────────────────────────────────────────────────

После отправки BotFather подтвердит регистрацию команд.

ОБЫЧНЫЙ ПОИСК (/search):
  /search Python разработчик
  /search Frontend junior Москва
  /search Backend middle СПб 200000

УМНЫЙ ПОИСК (/smart):
  /smart Хочу удаленную работу python от 150000
  /smart Ищу начальную позицию в IT с гибким графиком
  /smart Работа frontend middle в Москве
  /smart Нужна подработка для студента

КАЛЬКУЛЯТОР (/calc):
  /calc 200000 * 0.87
  /calc (150000 - 20000) / 12
  /calc 2 + 2
